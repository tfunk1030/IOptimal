Flow

Three steps

  1. Drop your .sto files — Drag one or two setup files from Documents\iRacing\setups into the browser.
  2. setupdelta decodes both — The files are parsed and matched field by field. No account required.
  3. Read the diff — Scan the table for changed values. Filter to show only differences, or search for a specific setting.

Questions

Common questions

What files do I need?

The .sto files from your iRacing setups folder. That's it.

Do I need to run telemetry first?

No. setupdelta reads the setup files directly. No laps, no telemetry app, no export from the garage.

Do I need to install anything?

No. It runs in your browser.

Can I compare before opening iRacing?

Yes — that's the point. Check the diff, then decide whether to load the setup in the sim.
